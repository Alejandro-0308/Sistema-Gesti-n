//Descripción

Sistema desarrollado en Java Swing para la gestión de insumos médicos, proveedores, empleados y movimientos de stock.

//Tecnologías utilizadas

Java 17
Java Swing
Apache Ant
Persistencia CSV

//Requisitos:

Java 17 o superior instalado.

Ejecutar:

SistemaInventarioMedico.jar

o desde consola:

java -jar SistemaInventarioMedico.jar
